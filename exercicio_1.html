A Linguagem HTML

HTML (abreviação para expressão inglesa HyperText Markup Language, que significa Linguagem de marcação de Hipertexto), é uma linguagem de marcação utilizada na construção de páginas na Web. Ducomentos HTML podem ser interpretados por navegadores.
